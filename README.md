# snowflake-ascii-art
Animates randomized snowflakes for a series of winter holiday ascii art drawings within a terminal window.

Add your own holiday ascii art inside the slides directory.

To install and run:

```bash
$ pip install termcolor
$ cd snow
$ python snow.py
```
